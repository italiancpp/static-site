<p style="font-weight: 400; text-align: justify;">
  Vcpkg appeared in 2016 as an open source C/C++ library manager with an expansive catalog of open source libraries that are tested together for compatibility. More recently, the team has been developing new features based on user feedback, including features relevant to enterprises with complex package management needs. This talk will give an overview of the tool and how to use it in a real-world scenario at your job, using new features we are planning on our roadmap:Â <a href="https://aka.ms/vcpkg/roadmap" data-saferedirecturl="https://www.google.com/url?q=https://aka.ms/vcpkg/roadmap&source=gmail&ust=1589530952270000&usg=AFQjCNFzkI0loWG-zNQwZjB92Uwj8Gdgeg">https://aka.ms/vcpkg/roadmap</a>.
</p>
